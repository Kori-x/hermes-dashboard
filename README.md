# Hermes Dashboard

Real-time dashboard for monitoring Hermes agent sessions, tool usage, and activity.

## Features

- Live agent session monitoring with phase tracking (processing, idle, awaiting input, needs approval)
- Activity feed showing tool calls, messages, and approvals across all sessions
- Per-session detail views with token usage, cost tracking, and tool breakdowns
- Session timeline visualization

## Stack

React + TypeScript + Vite

## Getting started

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173`.

## Build

```bash
npm run build
```

Output goes to `dist/`.
